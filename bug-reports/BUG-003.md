# BUG-003 — Address autocomplete shows duplicate suggestions

**Linear ID:** DOG-31  
**Severity:** Medium  
**Priority:** Medium  
**Status:** Fixed ✅  
**Platform:** iOS & Android  
**Component:** Create.tsx — address autofill

---

## Description

When a user types "Iso Omena" in the address autocomplete field on the Suggest a Venue screen, the same address appears 5 times in the suggestion list instead of once.

---

## Steps to Reproduce

1. Log in and navigate to "Suggest a Venue"
2. Tap the address input field
3. Type "Iso Omena"
4. Observe the suggestion dropdown

---

## Expected Result

Each unique address appears only once in the suggestions list.

## Actual Result

The same address string appears 5 consecutive times.

---

## Root Cause

The autocomplete API returns one result per business at a location, not per unique address. Iso Omena shopping centre has multiple businesses sharing the same address, so the API returned 5 results with identical address strings. No deduplication was applied before rendering.

## Fix Applied

Deduplication logic added to filter suggestions by unique address string before the list is rendered.
